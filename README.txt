# RSA-Java
A Java program that handles RSA message ciphering along with related commands using a graphical user interface.


INPUT FILE:
If the user wishes to provide their own input file for prime number generation,
the file name must be
primeNumbers.rsc

The cipher default file names are
privateKeyFile.xml
publicKeyFile.xml

The default blocked file name is
blockedFile.txt

The default unblocked file name is
unblockedFile.txt

Prime number checking done using 
https://en.wikipedia.org/wiki/Primality_test#Pseudocode


In the base prime numbers file, the primes are 11 digits, I could not find longer sizes, so they will need to be
provided by the user if larger are desired.
